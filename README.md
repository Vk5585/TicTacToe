Overview
Classic 3x3 Tic-Tac-Toe implemented using Java.

Supports two-player mode (Player X vs Player O) on the same device.

Clear visual indication of the current player and game state.

Features
Interactive board with clickable cells for making moves.

Automatic win and draw detection with game-over message.

Option to restart/reset the game without restarting the program.

Clean, beginner-friendly code structure, ideal for learning Java basics.
